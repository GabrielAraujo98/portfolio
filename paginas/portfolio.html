HTML5 - Resumo

HTML5 é uma linguagem de marcação de tecnologia central da Internet usada para estruturar e apresentar conteúdo para a World Wide Web. Em outubro de 2014, esta é a quinta revisão final e completa do padrão HTML do World Wide Web Consortium (W3C). A versão anterior, HTML 4, foi padronizada em 1997. HTML5 é uma ferramenta na categoria linguagens de uma pilha de tecnologia.


Prós

1º) Totalmente atualizado.
2º) Documentação muito detalhada.
3º) Suporte grande da comunidade.

Contras

1º) Precisa de CSS para deixar o site mais agradável.
2º) Precisa de Javascript para interação com o usuário.
3º) Limita-se a uma tecnologia apenas de marcação de hipertextos.

CSS - Resumo

CSS3 é a mais recente evolução da linguagem Cascading Style Sheets e visa estender o CSS2.1. Traz muitas novidades há muito esperadas, como cantos arredondados, sombras, gradientes, transições ou animações, além de novos layouts como multicolunas, caixas flexíveis ou layouts de grade. As partes experimentais são prefixadas pelo fornecedor e devem ser evitadas em ambientes de produção ou usadas com extremo cuidado, pois tanto a sintaxe quanto a semântica podem mudar no futuro. CSS 3 é uma ferramenta na categoria linguagens de uma pilha de tecnologia.


Prós

1º) Totalmente atualizado.
2º) Documentação muito detalhada.
3º) Suporte grande da comunidade.

Contras

1º) Não consegue fazer interassões diretas com o usuário.
2º) Precisa de Javascript para interação com o usuário.
3º) Precisa do HTML para poder ser utilizado nas páginas da internet.

React - Resumo

React é uma ferramenta na categoria Javascript UI Libraries de uma pilha de tecnologia. React é uma ferramenta de código aberto com 220,3 mil estrelas do GitHub e 45 mil forks do GitHub. Javascript - Resumo

JavaScript é mais conhecido como linguagem de script para páginas da Web, mas também é usado em muitos ambientes que não são de navegador, como node.js ou Apache CouchDB. É uma linguagem de script multiparadigma baseada em protótipo que é dinâmica e oferece suporte a estilos de programação orientados a objetos, imperativos e funcionais. JavaScript é uma ferramenta na categoria Idiomas de uma pilha de tecnologia.


Prós

1º) Pode ser utilizado em Front-end e no Back-end.
2º) Está em todos os lugares.
3º) Leve e de execução rápida.

Contras

1º) Segurança do lado do client.
2º) Variação do suporte dos navegadores.
3º) Código sempre visível para qualquer pessoa.
